Mantenimiento preventivo de equipos

1. Limpieza de temporales

Ejecutar Temp, %temp% y prefecth y borrar todo su contenido.

2. Liberar espacio (Disco local (C:))

Clic derecho sobre el disco local C - Propiedades - General - "Liberar espacio".

Tambien es accesible con el comando CLEANMGR dentro del CMD.

3. Configurar núcleos de arranque

Ejecutar comando MSCONFIG - Arranque - Opciones avanzadas - Número de procesadores (Máximo).

Ejecutar comando MSCONFIG - Arranque - Opciones avanzadas - Cantidad máxima de memoria (Disabled)

4. Apariencia para mejor rendimiento

Windows + pause - Configuración avanzada del sistema - Rendimiento (Configuración) - Seleccionar(Ajustar para obtener el mejor rendimiento).

5. Aplicaciones de arranque de windows

Ejecutar comando MSCONFIG - Inicio de windows - Deshabilitar aplicaciones con consumo alto o inútiles.

6. Borrar registros del visor de eventos

Ejecutar EVENTVWR y vaciar registros (Registros de Windows).

7. Desfragmentar disco

Abrir desfragmentador de disco y ejecutar análisis y optimización.

Tambien es accesible con el comando DEFRAG dentro del CMD.

8. Detección estado del disco

Ejecutar CrystalDiskInfo para determinar el estado del disco duro principal.

Disco bueno
Bueno

El estado del disco es bueno, no es causa de mal rendimiento.

Disco en riesgo
Riesgo

Contiene sectores defectuosos, puede afectar el rendimiento.

Disco dañado
Malo

Daño total, el disco requiere cambio.
